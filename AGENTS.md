# KubeCon LLM K8s Template

A Slidev presentation about "Taming Dependency Chaos for LLM in K8s", presented at KubeCon 2025 China.

## Tech Stack

- **Framework**: Slidev
- **Language**: Vue 3 / Markdown
- **Package Manager**: npm
- **Build Output**: `dist`

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy the presentation to Vercel or Netlify |

## Quick Start

```bash
npm install
npm run dev
```

## Special Features

- Rich icon library support (Carbon, Logos, Simple Icons, etc.)
- Twemoji for emoji rendering
- Remote presentation mode enabled

## Customization

- Edit `slides.md` to modify the presentation content
- Slidev uses Markdown with Vue components for slides
- See [Slidev documentation](https://sli.dev) for more features
