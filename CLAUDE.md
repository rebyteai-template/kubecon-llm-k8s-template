# AI Agent Instructions for KubeCon LLM K8s Template

This file provides guidance for AI coding agents customizing this presentation template.

## Quick Start

1. **Content**: Edit presentation content files
2. **Styling**: Customize styles as needed
3. **Assets**: Add images and diagrams

---

## Files to MODIFY (Customize These)

### Content
| Location | Purpose |
|----------|---------|
| Slide files | Presentation content |

### Assets
| Location | Purpose |
|----------|---------|
| `public/` or assets folder | Images, diagrams |

## Build and Test

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

---

## Deployment Notes

Deploy the build output folder to any static hosting service.
